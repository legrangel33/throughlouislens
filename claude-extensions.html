Claude Extensions: Chrome and Excel

6 min read · Updated May 2026 · throughlouislens

What Is a Claude Extension?

Claude extensions are first-party integrations that let Claude work directly inside tools you already use, your browser and your spreadsheets. Instead of copying and pasting between Claude and another app, the extension puts Claude inside the app.

Under the hood, most of these run on something called MCP (Model Context Protocol), a standard Anthropic built that lets Claude connect to external tools and data sources in a structured, secure way. You don't need to understand MCP to use these extensions, but it's worth knowing: it's why Claude can actually read your spreadsheet cells, click links in your browser, or navigate a Word document rather than just generating text you paste in yourself.

The two extensions:

  • Browser extension, Claude in Chrome lives in a side panel and can see and interact with whatever you're browsing.
  • Microsoft 365 add-in, Claude in Excel loads inside Excel itself as a task-pane add-in.

Claude in Chrome

What It Is

Claude in Chrome is a browser extension that opens as a side panel while you browse. It can see the page you're on, click links, fill forms, navigate tabs, and run multi-step workflows, without you switching windows or copying text.

How to Set It Up

  1. Go to the Chrome Web Store and search for Claude (by Anthropic).
  2. Click Add to Chrome and confirm the permissions prompt.
  3. Pin the extension: click the puzzle piece icon in your toolbar, then the pin next to Claude.
  4. Sign in with your Claude account.
  5. Click the Claude icon to open the side panel, it stays visible while you browse.

What It Can Do

Read and act on any page. Ask Claude to summarise a long article, extract key facts from a web page, or compare two pages you have open side by side. Claude reads the live page content, not a screenshot, so it gets the real text.

Fill out forms and click buttons. You can tell Claude to navigate a workflow for you, filling in a form, clicking through a multi-step process, or repeating a task you've done manually before.

Record and replay workflows. Click the record icon in the panel, walk through a task yourself, stop recording, and save it. Claude learns the steps and can repeat them on demand. Useful for anything repetitive, weekly reporting dashboards, data lookups, form submissions.

Scheduled browser tasks. Click the clock icon in the panel to set a recurring task, daily, weekly, or monthly. Claude will run it automatically in the background.

Multi-tab workflows. Drag tabs into Claude's designated tab group and it can view and interact with all of them at once, useful for cross-referencing sources, running comparisons, or batch-processing similar pages.

Ask Before Acting mode. Claude builds a plan for your approval first, then executes it step by step. It pauses to ask permission if it hits something outside the original plan. Good default for anything involving form submissions or account actions.

Practical Use Cases

  • Summarise a long research article while keeping it open for reference
  • Extract a table of data from a web page into plain text
  • Fill in a recurring form (expense reports, content submissions, vendor portals)
  • Monitor a page for changes on a schedule
  • Compare two product pages side by side and get a plain-language summary

Safety Notes

  • Claude only acts when you ask it to (unless you have set a scheduled task).
  • Review the plan in Ask Before Acting mode before letting Claude execute anything on a site that involves payments, account changes, or personal data.
  • Be thoughtful about granting Claude access on sites where you're logged into sensitive accounts, banking, health, legal.
  • You can revoke Chrome permissions at any time via Settings > Extensions.

Claude in Excel

What It Is

A Microsoft 365 add-in that loads Claude inside Excel. Claude can read your spreadsheet data, write formulas, build analysis, and generate charts, all from a panel inside Excel itself.

How to Set It Up

  1. Open Excel (Microsoft 365 desktop app or Excel online).
  2. Go to Insert > Add-ins > Get Add-ins.
  3. Search for Claude in the Office Add-ins store.
  4. Click Add and follow the sign-in prompt with your Claude account.
  5. Claude opens in a task pane on the right side of your spreadsheet.

What It Can Do

Write and explain formulas. Describe what you want in plain English, "calculate the 3-month rolling average of column B", and Claude writes the formula. It can also explain any formula already in your sheet in plain language, which is genuinely useful when you inherit a spreadsheet someone else built.

Analyse data. Select a range and ask Claude to find patterns, flag outliers, summarise trends, or compare periods. It reads the actual cell values.

Build pivot tables and charts. Tell Claude what you want to visualise and it can set up the structure for you, including chart type recommendations based on your data.

Clean messy data. Inconsistent date formats, mixed text/number columns, duplicate rows, blank cells, describe the problem and Claude suggests or applies the fix.

Explain what a spreadsheet is doing. Open a model someone else built and ask Claude to walk you through the logic, what each sheet does, what inputs drive what outputs, where the key assumptions live.

File uploads. You can upload a file for Claude to reference alongside your open spreadsheet, a CSV, another workbook, a data export.

Practical Use Cases

  • Fix a broken VLOOKUP or INDEX/MATCH without Googling the syntax
  • Summarise a month of transaction data into plain-language insights
  • Set up a dynamic budget tracker from raw expense rows
  • Audit a formula-heavy model before presenting it
  • Clean a data export before importing it elsewhere

Safety Notes

  • Claude reads your cell data within your active session. It does not store your spreadsheet data after the session ends.
  • Be mindful with confidential financial data, confirm your organisation's policy on Claude for Microsoft 365 before adding sensitive data to the context.
  • Claude can suggest changes but won't edit your spreadsheet without you approving and applying the change.

General Safety Principles

Permissions are scoped. Each extension only has access to what you're working on in that session. Claude in Chrome does not read your spreadsheets. Claude in Excel does not see your browser tabs.

You stay in control. Claude suggests, edits, and generates, but changes to your spreadsheet or any action in Chrome require you to accept them. Nothing is edited silently.

Sensitive data. For personal or small-project use, these tools are safe to use with everyday content. For confidential business data, financial models, legal documents, client-sensitive material, check whether your organisation has approved Claude for Microsoft 365 before connecting it to that content.

Account security. All Claude extensions authenticate through your Claude account. Use a strong password and enable two-factor authentication on your Anthropic account.

Scheduled and automated tasks. If you set up recurring Chrome tasks, review them periodically. Automations can interact with sites on your behalf, make sure they're still doing what you intended.

The Big Three Ecosystems

12 min read · Published May 2026 · throughlouislens

Back when the internet was new, you didn't just "use the internet." You used a browser. And depending on which one you picked, you got a different experience of the same underlying thing. Chrome was fast and Google-integrated. Firefox was open and privacy-focused. Safari was Apple-native. Internet Explorer was the one you opened once to download the others.

We're in that same moment now, but for AI.

Claude, ChatGPT, and Gemini are not just different LLM's. They are three competing operating systems for how AI plugs into your work and your computer. Each one has a desktop app, a terminal tool, a way to automate your browser, a research mode, a document editor, a file integration layer, and a developer platform. The terminology is completely different across all three, which makes comparing them feel impossible.

This guide is the Rosetta Stone. It maps every product category across all three providers so you know exactly what you're looking for, what it's called, and how to find it.

The Chat Interface: The Main Product

The face of each AI. The thing your parents have heard of.

All three work the same way in their simplest form: you type, it responds. The differences show up the moment you go one layer deeper. Claude's chat is known for longer context windows and more careful reasoning. ChatGPT reportedly has the widest user base (over 900 million users as of May 2026 with $25B ARR). Gemini is the most deeply integrated into Google's broader tech stack: Gmail, Docs, Drive.

The Desktop Agent: AI That Runs While You Work

This is where the product war is actually being fought in 2026. Each provider has a version of "AI that lives on your desktop, manages files, runs scheduled tasks, and ships work back to you."

ClaudeOpenAIGoogle
Claude CoworkChatGPT (Agent Mode)Gemini Spark (just launched)
Available on Claude desktop appBuilt into ChatGPT desktop for Plus/ProBeta rolling out to Ultra subscribers now

Claude Cowork is a standalone desktop app with a dedicated workspace folder on your machine. Claude reads your files directly, runs scheduled tasks in the background, and delivers finished outputs (documents, spreadsheets, presentations) to a folder you own. It has connectors to dozens of external tools (Slack, Notion, GitHub, etc.), and supports Skills (reusable instruction sets) and Plugins (community-built tool bundles).

ChatGPT Agent Mode evolved from what was called "Operator" in early 2025. It's now built directly into ChatGPT for Plus and Pro subscribers. It can navigate websites, run code, build spreadsheets and slideshows, and fill forms, all from the same interface you use for regular chat.

Gemini Spark was just announced at Google I/O on May 20, 2026. It's Google's answer to both: a 24/7 agentic assistant with out-of-the-box integrations to Gmail, Docs, and the entire Workspace suite. It runs on Google Cloud virtual machines (so it works even when your laptop is closed), and can interact with Chrome and the web. Early partner integrations include Canva, OpenTable, and Instacart. MCP support means more tools are coming. Still in beta for Ultra subscribers in the US.

The key difference: Cowork gives you a local file workspace and is the most mature product. Spark has the deepest first-party app integration (because Google owns Gmail, Docs, and Chrome). ChatGPT Agent Mode is the most seamlessly embedded into an existing conversation product.

The Terminal / CLI: AI in Your Command Line

All three providers have a coding agent you install via your terminal and run from the command line.

ClaudeOpenAIGoogle
Claude CodeCodex CLIGemini CLI (transitioning to Antigravity CLI)
npm install -g @anthropic-ai/claude-code npm install -g @openai/codex npx @google/gemini-cli

Not a technical person? Installing a CLI means running commands directly on your device. Proceed carefully. The commands above are a starting point, not a complete guide. Each provider has full installation documentation linked above. This guide is for informational purposes only.

Claude Code is Anthropic's CLI agent. It reads and edits your codebase, runs commands, and operates within your terminal with your files and git history directly accessible. Built for developers who want AI deeply embedded in their workflow, not as a web interface, but as a collaborator running in the same directory as your code.

Codex CLI is OpenAI's equivalent. As of April 2026, it reportedly has over 75,000 GitHub stars and 14.5 million monthly npm downloads. Recent updates added Goals (persistent task tracking), browser verification, automatic approval reviews, and plugins. It's the most widely adopted of the three.

Gemini CLI is Google's open-source CLI agent. It was the most generous on pricing: free with a personal Google account, with 60 model requests per minute and 1,000 per day. However, Google just announced at I/O 2026 that Gemini CLI is transitioning to Antigravity CLI as part of a broader agent-first developer platform. Free access ends June 18, 2026, paid tiers (Gemini Code Assist Standard/Enterprise) retain access. If you're a free user today, you have until June 18.

Computer Use / Desktop Automation: AI That Navigates

All three have a mode where the AI can take control of your screen: clicking buttons, filling forms, navigating interfaces that don't have APIs.

ClaudeOpenAIGoogle
DispatchCodex Computer UseGemini Spark (browser control)

Claude Dispatch is what Anthropic calls its computer-use mode. Claude sees your screen, moves your cursor, and operates any application as if it were a person sitting at your desk. Designed for repetitive tasks in apps that have no API: form-filling, navigating legacy software, and so on. Slow by design, you pay attention to it.

Codex Computer Use (released as "Codex for almost everything") lets Codex operate macOS applications autonomously using its own cursor. Multiple Codex agents can run in parallel in the background without disrupting your foreground work. This is currently one of the most capable implementations.

Gemini Spark will include browser control through Chrome, with the ability to send texts and emails and operate your browser directly. This feature is coming in the weeks after launch, not fully available at time of writing.

Projects: Persistent Workspaces

A "project" is a saved context: a place where you store files, instructions, and conversation history related to a specific ongoing effort.

ClaudeOpenAIGoogle
Claude ProjectsChatGPT ProjectsNotebooks in Gemini / NotebookLM

Claude Projects let you give Claude persistent memory, reference files, and custom instructions per project. The AI retains context across sessions within that project.

ChatGPT Projects work the same way: group related chats, upload reference files, add custom instructions. The AI stays on-topic to the project's context window.

Google's answer is two-pronged: Notebooks in Gemini is a project management tool synced with NotebookLM. NotebookLM is Google's source-grounded research assistant: you give it documents, it reasons only from those. Gemini Deep Research is now integrated directly inside NotebookLM ↗.

Document Creation: AI That Generates

Beyond generating text in a chat window: AI that produces actual editable documents, spreadsheets, presentations, and interactive dashboards.

ClaudeOpenAIGoogle
Claude Skills (Word, Excel, PPT, PDF) + Live ArtifactsChatGPT CanvasDocs Live + Google Workspace AI

Claude's approach separates two things. Skills (Word, Excel, PowerPoint, PDF) handle formatted document creation: real .docx, .xlsx, .pptx files delivered to your folder. Live Artifacts (launched April 21, 2026) are interactive HTML pages that re-execute and pull fresh data every time you open them. Think of a dashboard that's connected to your live tools, not a static export.

ChatGPT Canvas is OpenAI's collaborative document editor inside ChatGPT. You write alongside the AI in a shared editor pane. As of 2026, Canvas can also build interactive apps and call APIs, making it more powerful than a simple document editor.

Google Docs Live was announced at I/O 2026: a conversational interface for creating and editing documents directly in Docs, with Gemini integrated. Rolling out to Android and iOS for AI Pro and Ultra subscribers.

Image Generation: AI That Designs

ClaudeOpenAIGoogle
Claude DesignGPT-Image-2 (formerly DALL-E)Nano Banana

Claude takes a different approach. Instead of a raster image-generation model, Anthropic ships Claude Design, a dedicated product for producing interfaces, slides, diagrams, and interactive visualizations directly through Artifacts. Launched as a research preview on April 17, 2026 and powered by Claude Opus 4.7. Its image analysis is still considered best-in-class. For raster image output, Claude can route through external tools via MCP, but the native focus is on design and structured visuals rather than photorealistic rendering.

OpenAI retired DALL-E 3 in March 2026 (fully removed from the API on May 12, 2026). The current model is GPT-Image-2 (launched April 21, 2026), now the default inside ChatGPT. It introduced reasoning before generation, 4K resolution, and multilingual text rendering. Pricing is token-based.

Google has the most developed image generation lineup. Nano Banana is Google's family of image generation and editing models, built on the Gemini image stack:

  • Nano Banana (original), launched August 2025
  • Nano Banana Pro, built on Gemini 3 Pro, integrates with Google Search knowledge for real-world accuracy
  • Nano Banana 2, launched February 2026 (technically Gemini 3.1 Flash Image), now the default in Google Search, Lens, and AI Mode across 141 countries

Nano Banana supports up to 4K resolution, multi-image blending, and character consistency across up to 5 characters. All outputs carry a SynthID watermark for AI transparency.

The key difference: Google is the clear leader here with a mature, multi-tier product embedded across its ecosystem. OpenAI has a capable replacement in GPT-Image-2 after transitioning away from DALL-E.

Deep Research: AI That Dives Deep

Multi-step research mode where you ask a big question and the AI autonomously browses the web, synthesizes sources, and returns a full report.

ClaudeOpenAIGoogle
Claude (research mode)ChatGPT Deep ResearchGemini Deep Research / NotebookLM

ChatGPT Deep Research was one of the defining product launches of 2025. The AI finds, analyzes, and synthesizes hundreds of online sources to produce a comprehensive report at "research analyst" level. A more capable version is live now. Available on Plus, with higher limits on Pro/Enterprise.

Gemini Deep Research is now embedded inside NotebookLM, combining Google's search grounding with source-cited research. You can run deep research and immediately have the sources in your Notebook for follow-up reasoning.

Browser Extension: AI Inside Your Browser

ClaudeOpenAIGoogle
Claude in Chrome ↗ (extension) ChatGPT Atlas ↗ (standalone AI browser) Gemini (native to Chrome)

This is one of the areas where Claude and Google go different directions. Claude in Chrome is a browser extension that lets you bring Claude into any webpage: summarizing content, drafting replies, and running Claude while you browse. You install it from the Chrome Web Store.

Google doesn't rely on an extension because Gemini is built directly into Chrome as a persistent side panel. As of January 2026, it includes Auto Browse, an agentic mode that can scroll, click, fill forms, and navigate websites on your behalf, making it reactive, not just ambient. Available to AI Pro and Ultra subscribers in the US.

OpenAI took a different approach entirely: ChatGPT Atlas is a standalone Chromium-based browser (launched October 2025, macOS first) with ChatGPT built directly into the sidebar, an agent mode that can operate websites on your behalf, and browser memory across sessions. Rather than plugging into Chrome, OpenAI built their own browser around ChatGPT.

A note on browser tools: Installing extensions or dedicated AI browsers carries inherent security risks. Only install from official sources, review the permissions you grant, and proceed at your own discretion.

Workspace Extension: AI Inside Your Spreadsheet

ClaudeOpenAIGoogle
Claude in Excel (extension)ChatGPT for Excel (Microsoft partnership)Gemini in Google Sheets (native)

Claude in Excel is an add-in that brings Claude into Microsoft Excel: writing formulas, analyzing data, and building out spreadsheets through conversation. To install it, search for Claude by Anthropic in Microsoft AppSource ↗ or go to Home > Add-ins directly inside Excel and search from there. Requires a paid Claude plan (Pro, Max, Team, or Enterprise) and works on Excel for web, Windows (Microsoft 365), and Mac (version 16.46 or later).

ChatGPT for Excel exists through Microsoft's broader Copilot and OpenAI partnership within Microsoft 365.

Gemini in Google Sheets is native. Gemini is already inside every Google Workspace product.

Connectors / Integrations: How AI Plugs Into Your Tools

ClaudeOpenAIGoogle
Claude ConnectorsChatGPT App IntegrationsGoogle MCP / Workspace

Claude Connectors is Anthropic's directory of third-party tool integrations available inside Cowork and Claude. Current list includes GitHub, Slack, Gmail, Google Drive, Notion, Asana, Linear, Jira, Figma, Stripe, Canva, ClickUp, and dozens more. All connections use MCP (Model Context Protocol).

ChatGPT App Integrations launched with pilot partners including Figma, Canva, Booking.com, Coursera, Expedia, Spotify, and Zillow. Available to all logged-in users on Free, Go, Plus, and Pro plans (outside the EEA, Switzerland, and UK for now).

Google has the home-team advantage in Workspace integrations (Gmail, Drive, Docs, Sheets, Calendar are all native). External tools connect via Google's MCP implementation. Gemini Spark launched with Canva, OpenTable, and Instacart as early partners.

A note on terminology: MCP is the protocol, the plumbing that makes connections possible. A Connector is a single tool integration built on MCP. A Plugin bundles multiple Connectors and Skills into one installable package. They're related but not the same thing.

Skills / GPTs / Gems: Reusable AI Configurations

Saved AI setups tuned for a specific job: a customer support persona, a coding reviewer, a specific document style.

ClaudeOpenAIGoogle
Skills + PluginsGPTs (Custom GPTs)Gems

Claude Skills are instruction bundles: pre-built workflows for specific tasks (Word document creation, Excel modelling, social media captions, etc.). You can install community Skills or build your own. Plugins are larger bundled tools with MCPs and Skills combined, installable from a plugin marketplace.

Custom GPTs (OpenAI) are user-created or community-created AI configurations that anyone can publish and share. The GPT Store has millions of GPTs. Think of them as apps built on top of ChatGPT.

Gems (Google) are Gemini's equivalent: saved AI personas with custom instructions and tools attached.

The Developer Platform: Building On Top

ClaudeOpenAIGoogle
Anthropic API + Claude Agent SDKOpenAI API + Agent SDKGoogle AI Studio + Gemini API + Antigravity SDK

All three have full APIs for building products on top of their models.

The more interesting layer in 2026 is the agent SDK: tools for building multi-step autonomous AI agents.

Claude Agent SDK is what powers Cowork under the hood: a framework for building agents that can use tools, manage memory, and execute multi-step workflows. Available to developers to build Cowork-style functionality into their own products.

OpenAI's Agent SDK is the foundation of the Codex agent platform. Supports tool use, parallel agents, and code execution.

Google's Antigravity SDK is the latest. Announced at I/O 2026, it gives developers programmatic access to the same agent harness powering Google's own products, co-optimized for Gemini models, self-hostable.

How to Think About It

Each provider has a clear centre of gravity.

Anthropic (Claude) is building for people who want AI embedded in serious work: document creation, project management, scheduled automation, and creative professional workflows. Cowork is the hub. Everything connects back to a workspace folder you own.

OpenAI (ChatGPT) is building for the widest possible audience and the most deeply agentic product. Codex Computer Use running multiple agents in parallel while you work is genuinely impressive. The GPT Store and massive third-party ecosystem mean there's a ChatGPT for almost every niche task.

Google (Gemini) is building for people already inside Google's world. If you live in Gmail, Google Docs, and Google Calendar, Gemini Spark is the most seamless path because it doesn't have to ask for permission to access any of it. The Antigravity SDK and Gemini Spark represent a major bet on agents announced this week. Google I/O 2026 was less about models and more about turning every Google product into an agent.

The browser wars ended with Chrome winning. This one is still wide open.

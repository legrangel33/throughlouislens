Big Three
Ecosystems.

Claude, ChatGPT, and Gemini are not just different LLMs. They are three competing operating systems for how AI plugs into your work and your computer.

Each one has a desktop app, a terminal tool, a way to automate your browser, a research mode, a document editor, a file integration layer, and a developer platform. The terminology is completely different across all three. This guide is the Rosetta Stone.

GPT-5.5 Pro / Thinking

Google
Gemini 3.1 Pro
+ Gemini 3.5 Flash
3.5 Flash launched at I/O 2026 — 4× faster than other frontier models. Gemini 3.5 Pro arrives next month.

Google also launched Gemini Omni at I/O 2026 — a model that takes image, audio, video, and text as input and outputs video. OpenAI's equivalent was Sora, but Sora was discontinued in April 2026. Anthropic doesn't have a video-generation model.

///03
AI that runs while you work

The Desktop Agent.

This is where the product war is being fought in 2026. Each provider has a version of "AI that lives on your desktop, manages files, runs scheduled tasks, and ships work back to you."

Anthropic
Claude Cowork
Desktop app
Standalone desktop app with a dedicated workspace folder. Reads files directly, runs scheduled tasks, delivers outputs to a folder you own. Connectors, Skills, Plugins.
OpenAI
Agent Mode
Built into ChatGPT · Plus / Pro
Evolved from "Operator." Navigates websites, runs code, builds spreadsheets and slideshows, fills forms — from the same interface you chat in.
Google
Gemini Spark
Beta · Ultra subscribers
Announced at Google I/O May 20, 2026. 24/7 agentic assistant with Gmail, Docs, Workspace integrations. Runs on Google Cloud VMs — works when your laptop is closed.
/ The key difference

Cowork gives you a local file workspace and is the most mature product. Spark has the deepest first-party app integration (because Google owns Gmail, Docs, and Chrome). Agent Mode is the most seamlessly embedded into an existing conversation product.

AI in your command line

Terminal · CLI.

All three providers have a coding agent you install via your terminal and run from the command line.

Anthropic
Claude Code

Reads and edits your codebase, runs commands, operates inside your git history.
OpenAI
Codex CLI
npm install -g

75K+ GitHub stars, 14.5M monthly npm downloads. Goals, browser verification, plugins. Most widely adopted of the three.
Google
Antigravity CLI

///05
AI that navigates

Computer Use.

All three have a mode where the AI takes control of your screen — clicking buttons, filling forms, navigating interfaces that don't have APIs.

Anthropic
Dispatch
Sees your screen, moves your cursor, operates any application. Designed for repetitive tasks in apps without an API. Slow by design — you pay attention to it.
OpenAI
Codex Computer Use
Operates macOS apps autonomously with its own cursor. Multiple Codex agents can run in parallel without disrupting foreground work. Currently one of the most capable implementations.
Google
Gemini Spark
Browser control through Chrome, with the ability to send texts and emails and operate your browser directly. Coming in the weeks after Spark's launch.
///06
Persistent workspaces

Projects.

A "project" is a saved context — a place where you store files, instructions, and conversation history related to a specific ongoing effort.

Anthropic
Claude Projects
Persistent memory, reference files, and custom instructions per project. Context retained across sessions.
OpenAI
ChatGPT Projects
Group related chats, upload reference files, add custom instructions. The AI stays on-topic to the project context.
Google
Notebooks · NotebookLM
Two-pronged. Notebooks in Gemini is project management synced with NotebookLM — source-grounded research with Gemini Deep Research integrated.
///07
AI that generates

Document Creation.

Beyond generating text in a chat window — AI that produces actual editable documents, spreadsheets, presentations, and interactive dashboards.

Anthropic
Skills + Live Artifacts
Word · Excel · PPT · PDF
Skills handle formatted document creation — real .docx, .xlsx, .pptx files. Live Artifacts (April 21, 2026) are interactive HTML pages that pull fresh data every time you open them.
OpenAI
ChatGPT Canvas
Collaborative document editor inside ChatGPT. As of 2026, can also build interactive apps and call APIs — more powerful than a simple editor.
Google
Docs Live + Workspace AI
Docs Live (I/O 2026) — conversational interface for creating and editing documents directly in Docs. Rolling out to AI Pro and Ultra on iOS and Android.
///08
AI that designs

Image Generation.

Three providers, three completely different bets on what generative imagery is for.

Anthropic
Claude Design
Dedicated product for interfaces, slides, diagrams, and interactive visualizations through Artifacts. Research preview April 17, 2026, powered by Opus 4.7. Image analysis still considered best-in-class.
OpenAI
GPT-Image-2
Formerly DALL-E
DALL-E 3 retired March 2026. GPT-Image-2 (April 21, 2026) now default inside ChatGPT — reasoning before generation, 4K resolution, multilingual text rendering.
Google
Nano Banana
Nano · Pro · 2
Three tiers: original (Aug 2025), Pro (built on Gemini 3 Pro), and Nano Banana 2 (Feb 2026) — default in Search, Lens, AI Mode across 141 countries. 4K, multi-image blending, SynthID watermark.
/ The key difference

Google is the clear leader here with a mature, multi-tier product embedded across its ecosystem. OpenAI has a capable replacement in GPT-Image-2. Claude focuses on design and structured visuals over photorealistic rendering.

///09
AI that dives deep

Deep Research.

Multi-step research mode — ask a big question, the AI browses the web autonomously, synthesizes sources, returns a full report.

Anthropic
Research mode
Built into Claude. Synthesizes sources and structures a report alongside your conversation.
OpenAI
Deep Research
One of the defining product launches of 2025. Finds, analyzes, and synthesizes hundreds of online sources to produce a research-analyst-level report. Available on Plus, higher limits on Pro/Enterprise.
Google
Gemini Deep Research
Embedded inside NotebookLM. Combines Google's search grounding with source-cited research — run a query, immediately work with the sources in your Notebook.
///10
AI inside your browser

The Browser.

Three completely different bets — an extension, a side panel, or a whole new browser.

Anthropic
Claude in Chrome
Chrome extension
Brings Claude into any webpage: summarizing content, drafting replies, running Claude while you browse. Installed from the Chrome Web Store.
OpenAI
ChatGPT Atlas
Standalone browser · macOS
A Chromium-based browser (October 2025) with ChatGPT in the sidebar, an agent mode that can operate websites, and browser memory across sessions.
Google
Gemini · native in Chrome
Persistent side panel
Built directly into Chrome. Auto Browse (Jan 2026) is an agentic mode that can scroll, click, fill forms, and navigate websites for you. AI Pro and Ultra.
/ Note on browser tools

Installing extensions or dedicated AI browsers carries inherent security risks. Only install from official sources, review permissions, and proceed at your own discretion.

///12
How AI plugs into your tools

Connectors.

Live integrations to the third-party tools you already use.

Anthropic
Claude Connectors
GitHub, Slack, Gmail, Drive, Notion, Asana, Linear, Jira, Figma, Stripe, Canva, ClickUp, and dozens more. All connections use MCP.
OpenAI
App Integrations
Pilot partners include Figma, Canva, Booking.com, Coursera, Expedia, Spotify, Zillow. Available to all logged-in users on Free, Go, Plus, Pro (outside EEA, CH, UK for now).
Google
Google MCP + Workspace
Home-team advantage — Gmail, Drive, Docs, Sheets, Calendar are all native. External tools via Google's MCP. Spark launched with Canva, OpenTable, Instacart.
/ Terminology

MCP is the protocol — the plumbing that makes connections possible. A Connector is a single tool integration built on MCP. A Plugin bundles multiple Connectors and Skills into one installable package.

Saved AI setups tuned for a specific job — a customer support persona, a coding reviewer, a brand voice.

Instruction bundles for specific tasks. Install community Skills or build your own. Plugins bundle MCPs and Skills into one installable package from a plugin marketplace.

Google
Gems
Gemini's equivalent — saved AI personas with custom instructions and tools attached.

Building on top

Developer Platform.

All three have full APIs for building products on top of their models. The interesting layer in 2026 is the agent SDK — tools for building multi-step autonomous AI agents.

Anthropic
Claude Agent SDK
+ Anthropic API
Powers Cowork under the hood. Framework for building agents that use tools, manage memory, and execute multi-step workflows. Available to build Cowork-style functionality into your own products.
OpenAI
OpenAI Agent SDK
+ OpenAI API
Foundation of the Codex agent platform. Supports tool use, parallel agents, and code execution.
Google
Antigravity SDK
+ AI Studio · Gemini API
Announced at I/O 2026. Programmatic access to the same agent harness powering Google's own products. Co-optimized for Gemini models, self-hostable.

Centres of gravity

How to Think About It.

Anthropic / Claude
Serious work, embedded.

Building for people who want AI embedded in serious work: document creation, project management, scheduled automation, creative professional workflows. Cowork is the hub. Everything connects back to a workspace folder you own.

OpenAI / ChatGPT
Widest reach, most agentic.

Building for the widest possible audience and the most deeply agentic product. Codex Computer Use running multiple agents in parallel is genuinely impressive. The GPT Store and massive third-party ecosystem mean there's a ChatGPT for almost every niche task.

Google / Gemini
Already in your world.

Building for people already inside Google's world. If you live in Gmail, Docs, Calendar, Spark is the most seamless path — it doesn't have to ask for permission. Antigravity SDK and Spark represent a major bet on agents.

The browser wars ended with Chrome winning. This one is still wide open.
